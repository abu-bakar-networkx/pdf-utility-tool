# pdf-utility-tool
Python command-line tool for PDF manipulation (merge, split, rotate, encrypt/decrypt).

Tool functionalities you can use:-
- Read text from PDF files
- Merge multiple PDF files
- Rotate PDF pages
- Split PDF files (mid or custom range)
- Encrypt PDF with password
- Decrypt protected PDF

Requirements:- 
- Python 3.8 or higher
- Dependencies --> Can by installed using "pip install -r requirements.txt"

How to run:- 
1. Clone the repository
- git clone https://github.com/abu-bakar-networkx/pdf-utility-tool.git
2. Navigate to the Project Folder
- cd pdf-utility-tool
3. Run the Program
- python main.py or python3 main.py 

How to use this application:- 
After running the script, select the desired operation from the menu (1–6) and follow the prompts to perform actions on your PDF files.
Make sure your PDF files are in the same directory as main.py.

Notes:- 
- Ensure correct file names are provided when prompted
- PDF Files should be located in the same directory as the script
